# 🕒 Analog Clock

A simple and responsive **Analog Clock** built using **HTML, CSS, and JavaScript**. This project demonstrates how to use JavaScript to manipulate DOM elements and create real-time animations.

## 🚀 Features
- Real-time clock (hours, minutes, seconds)
- Smooth hand movement
- Responsive design
- Pure HTML, CSS, and JavaScript (no libraries)

## 🛠️ Technologies Used
- **HTML** – Structure of the clock  
- **CSS** – Styling and layout  
- **JavaScript** – Logic for updating time dynamically
